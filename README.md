# Financial Impulse Behaviour Detection

## Problem
Young adults often overspend due to emotional and impulsive decisions.
This project detects impulsive spending behaviour using behavioural analytics.

## Features
- Impulse Risk Score (0–100)
- Behavioural spending profiles
- Explainable risk triggers
- Personalized financial nudges

## Tech Stack
- Python
- Streamlit
- Pandas
- Matplotlib

## Deployment
Deployed using Streamlit Cloud.
